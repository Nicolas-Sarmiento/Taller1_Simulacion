# Taller1: Simulación - Números Pseudoaleatorios y Método MonteCarlo

Implementación de un **generador congruencial lineal (LCG)** con validación Hull-Dobell y el **método de MonteCarlo** para estimar áreas bajo curvas.

## 📚 Documentación

**➡️ Ver documentación completa en: `DOCUMENTACION_COMPLETA.md`**

Este archivo contiene:
- Descripción detallada del proyecto
- Explicación del algoritmo LCG
- Validación Hull-Dobell
- 4 pruebas estadísticas (Frecuencia, Tendencia, Correlación, Kolmogorov-Smirnov)
- Detección de números repetidos
- Método MonteCarlo completo
- Arquitectura y flujo de ejecución
- Fórmulas matemáticas
- Guía de solución de problemas
- Ejemplos de uso

## 🚀 Inicio Rápido

1. Abrir `Taller1_Simulacion.sln` en Visual Studio
2. Compilar: `Ctrl + Shift + B`
3. Ejecutar: `F5`
4. (Opcional) Ingresar parámetros (multiplicador, incremento, semilla)
5. Presionar "Ejecutar"
